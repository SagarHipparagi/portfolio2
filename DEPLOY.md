### Or Deploy via GitHub

1. **Initialize Git** (if not already done):
   ```bash
   cd C:\Users\sagar\.gemini\antigravity\scratch\portfolio-landing
   git init
   git add .
   git commit -m "Initial commit: Personal portfolio"
   ```

2. **Create GitHub Repository**:
   - Go to https://github.com/new
   - Create a new repository (e.g., "portfolio")
   - Don't initialize with README

3. **Push to GitHub**:
   ```bash
   git branch -M main
   git remote add origin https://github.com/SagarHipparagi/portfolio.git
   git push -u origin main
   ```

4. **Deploy on Vercel**:
   - Go to https://vercel.com
   - Click "New Project"
   - Import from GitHub
   - Select your portfolio repository
   - Click "Deploy" (Vercel auto-detects configuration)

### Or Drag & Drop Deploy
